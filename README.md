# 📊 Nexus CRM - Gestion Partenaires & Workflow

Nexus CRM est une application de gestion moderne développée avec **Laravel 11**, conçue pour centraliser les données des partenaires et suivre l'avancement des projets en temps réel.

## ✨ Fonctionnalités Clés
- **Tableau de Bord Dynamique :** Visualisation des statistiques (Revenus, Taux de réussite) avec **Chart.js**.
- **Gestion CRUD Complète :** Système robuste pour gérer les partenaires et leurs projets associés.
- **Recherche Instantanée :** Interface réactive et filtrage fluide grâce à **Alpine.js**.
- **Reporting PDF :** Génération de rapports professionnels optimisés pour l'impression.

## 🛠️ Stack Technique
- **Framework :** Laravel 11 (PHP 8.2).
- **Frontend :** Tailwind CSS & Alpine.js.
- **Base de données :** MySQL.
- **Visualisation :** Chart.js.

## 🚀 Installation
1. `git clone https://github.com/Siham143/nexus-crm.git`
2. `composer install`
3. `cp .env.example .env` (Configurez votre DB)
4. `php artisan migrate`
5. `php artisan serve`
